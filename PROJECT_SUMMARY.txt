================================================================================
SETUP INSTRUCTIONS
================================================================================

1. Update appsettings.json with your SQL Server connection details:
   "Server=YOUR_SERVER;Database=planning;User Id=USER;Password=PASSWORD;"

2. Restore NuGet packages:
   dotnet restore

3. Apply database migrations:
   dotnet ef database update

4. Build the solution:
   dotnet build

5. Run the application:
   dotnet run

6. Access at: https://localhost:7123 or http://localhost:5241
